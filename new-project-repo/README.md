# New Project Repo

This folder is a starter repository template for a new project.

## Quick start

1. Create a new GitHub repository (empty) with your desired name.
2. From this folder, initialize git and push:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-github-repo-url>
git push -u origin main
```

## Project structure

- `src/` – source code
- `.gitignore` – sensible defaults for common tooling
